# Dual-Pico-Based-Custom-Gaming-Controller-with-Posture-Monitoring-System
Custom gaming controller and ultrasonic posture system using Raspberry Pi Pico

## Overview
This project implements a custom controller using Raspberry Pi Pico and a posture monitoring system using an ultrasonic sensor.

## Features
- Joystick + button controller (USB HID keyboard)
- Ultrasonic distance detection
- LCD feedback system
